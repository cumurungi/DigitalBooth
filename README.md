# Advice for Marriage Digital Booth

A polished wedding guest experience for capturing advice, blessings, photos, voice notes, and short videos.

## Features

- Tablet-friendly guest booth UI with a premium wedding style
- Supports photos in addition to advice, blessings, voice notes, and short videos
- Live preview of entries and a memory book organizer
- Automatic category grouping for easy browsing after the wedding
- Local browser storage for the message list

## How to run

1. Open a terminal in `c:\Projects\DigitalBooth`
2. Run `npm install`
3. Run `npm run start`
4. Open the local URL shown in the terminal

## What to expect

- Guests can choose the type of message they want to leave and attach a photo, audio, or video file
- After the event, switch to the memory book view to see categories, item counts, and a gallery of your favorite messages
- The experience is designed to feel warm, elegant, and easy to use for your guests

## Wedding kiosk mode

Kiosk mode means the booth runs as a dedicated tablet or event station with a focused interface.
It keeps guests on the booth experience and makes it easy to capture messages without navigating away.

## AI-powered summary export

This future feature will organize your collected messages into a polished memory book.
AI can later group themes, summarize wishes, and create a shareable keepsake from the guest entries.
